# Vaquinha Solidária — Landing Page de Doações (PIX)

**Resumo**  
Landing page responsiva otimizada para celulares, criada para divulgar uma vaquinha solidária destinada a custear gastos médicos. A página foca em conversão (doação via PIX), apresenta espaço para fotos (hero + galeria), versículo inspirador e contato via WhatsApp. Design acolhedor com paleta em laranja pastel + verde pastel.

---

## Funcionalidades principais
- Exibição de **chave PIX** destacada (com botão "Copiar").
- Exibição de **QR Code** para doação rápida.
- **Galeria** responsiva para 3–6 fotos.
- Versículo inspirador em destaque.
- Botão flutuante do **WhatsApp** para contato direto.
- Animações suaves (fade-in, hover em imagens) e foco na acessibilidade.
- Estrutura de arquivos separada: `index.html`, `assets/css/styles.css`, `assets/js/main.js` e `assets/img/`.
